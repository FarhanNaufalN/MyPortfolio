Mini Waiter Robot Project

Project information

  • Category: Internet of Things
  • Client: University
  • Project Year: 2024

Prototype Mini Waiter Robot Based on Arduino Uno with Voice and Remote Control is a prototype of a waiter robot or waiter designed to be able to deliver food. There are two parts to this robot prototype, namely the bottom and the top. The bottom has 4 wheels that can be controlled by voice (Voice Control) and can move according to commands (forward, backward, turn right, turn left). Furthermore, the top is a part for storing food that has hands and has almost the same function as a tray or tray that can be controlled using a remote control and moves according to commands (forward and backward). The purpose of making this prototype is the basis of a waiter robot so that it can be developed into a waiter robot that has better performance and quality and is in accordance with market needs.

  • C++
  • 3D Blender
  • Arduino UNO
  • Voice Command
