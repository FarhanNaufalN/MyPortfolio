Project information

  • Category: Social Media
  • Client: Dicoding
  • Project Year: 2024

Story App is an Android application designed to allow users to upload and share their daily activities through personalized stories. It features registration and login functionalities to ensure user privacy and security, along with integration with Google Maps to display the location where the stories are created, enabling users to see the locations of their story friends in real-time. The app leverages APIs to communicate with the backend server, facilitating user registration, login, story uploads, and data retrieval in an efficient and structured manner. The Google Maps API is used to show the user's location on a map, providing contextual information about where the stories are made. The implementation follows Clean Architecture principles, dividing the application into distinct layers: Presentation, Domain, and Data. The Presentation Layer handles the user interface and presentation logic, the Domain Layer manages business logic and use cases, and the Data Layer is responsible for data retrieval and storage, ensuring that the application remains modular and easy to maintain. Animations play a significant role in enhancing the user experience in Story App. They are used to create smooth page transitions, provide visual feedback during loading and story posting, and add interactive effects to the map and other UI elements. These animations enrich user engagement and satisfaction by making interactions intuitive and enjoyable. Through this approach, Story App not only offers valuable features but also delivers a seamless and delightful user experience.
