Cancer Clasification App

Project information

  • Category: Medical App
  • Client: Dicoding
  • Project Year: 2024

Cancer Classifier is a mobile application developed using the Kotlin programming language and TensorFlow Lite (TF Lite) to classify cancer quickly and accurately. This application is designed to help doctors, researchers and individuals who want to carry out early detection and monitoring of cancer using medical data.
